    Дана строка. Если в этой строке более одного символа, выведите в консоль предпоследний символ этой строки.

    Решение

    string = str(input('Введите строку: '))

    len_str = len(string)

    if len_str == 1:

    print('некоректная запись, в ваше строке 1 элемент')

    else:

    print(f'Предпоследний символ: {string[-2]}')

    № 2

    Даны два целых числа. Проверьте, что первое число без остатка делится на второе.

    Решение

    num1 = int(input('Введите первое число: '))

    num2 = int(input('Введите второе число: '))

    rezalt = num1 % num2

    if rezalt == 0:

    print('Числа делятся без остатка')

    else:

    print(f'Числа делятся с остатком: {rezalt}')

    № 3

    Дана некоторая строка:

    'abcde'

    Получите список ее символов:

    ['a', 'b', 'c', 'd', 'e']

    Решение

    string = 'abcde'

    list = list(string)

    print(list)

    № 4

    Дан список:

    [1, 2, 3, 4, 5, 6]

    Получите из него следующий срез:

    [3, 4, 5]

    Решение

    list = [1, 2, 3, 4, 5, 6]

    a = list[2:5]

    print(a)
