    № 1

    Дано число. Выведите в консоль первую цифру этого числа.

    Решение

    num = int(input('введите число: '))

    num_str = str(num)

    first_char = num_str[0]

    print(f'первая цифра: {int(first_char)}')

    № 2

    Дано число. Выведите в консоль последнюю цифру этого числа.

    Решение

    num = int(input('введите число: '))

    num_str = str(num)

    first_char = num_str[-1]

    print(f'Последняя цифра: {int(first_char)}')

    № 3

    Дано число. Выведите в консоль сумму первой и последней цифры этого числа.

    Решение

    num = int(input('Введите число: '))

    num_str = str(num)

    first_char = num_str[0]

    last_char = num_str[-1]

    first_char = int(first_char)

    last_char= int(last_char)

    sum_last_first = first_char + last_char

    print(f'Сумма первой и последней цифры = {sum_last_first}')

    № 4

    Дано число. Выведите количество цифр в этом числе.

    Решение

    num = int(input('Введите число: '))

    num_str = str(num)

    len_str = len(num_str)

    print(f'Кол-во цифр в это числе: {len_str}')

    № 5

    Даны два числа. Проверьте, что первые цифры этих чисел совпадают.

    Решение

    num1 = int(input('Введите первое число: '))

    num2 = int(input('Введите второе число: '))

    num1_str = str(num1)

    num2_str = str(num2)

    first_num_one = num1_str[0]

    first_num_two = num2_str[0]

    if first_num_one == first_num_two:

    print(f'Первые числа ровны: {first_num_one} = {first_num_two}')

    else:

    print(f'Первые числа не ровны: {first_num_one} не = {first_num_two}')

    № 6

    Дан список:

    [1, 2, 3, 4, 5, 6]

    Получите из него следующий срез:
